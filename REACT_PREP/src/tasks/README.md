# React Machine Coding — Task Index

Practice log for React machine-coding interview rounds. Each folder below is
self-contained: a `README.md` problem statement, an `index.tsx` boilerplate
component, and a `styles.css` skeleton. Nothing is pre-solved.

## How to practice a task

1. Open the task folder, read its `README.md`, set a timer for the suggested time box.
2. Implement the logic inside `index.tsx` / `styles.css`.
3. Import the component into `../../App.tsx` (temporarily, like the existing
   `use-custom-effect` block) and run `npm run dev` to see it live.
4. Check it off below once you can build it cleanly within the time box.
